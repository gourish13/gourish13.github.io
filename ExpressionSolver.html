Expression Solver

Documentation :
  • Write expression in the box and get the result .
  • Don't forget to place brackets correctly to get accurate result .
    Only '( )' bracket supported .
  • You can select use custom variables to define your own variable for expression .
  • Only for solving simple arithmetic expression with following operators :
    +(add), -(subtract), *(multiply), /(divide) and %(modulas/remainder).
  • For X to the power n operation use X**n .
  • For nth root of X operation use X**(1/n) .
  • Get result in desired base y setting base value to the base of desired number system (default = 10 Decimal Number System) .
